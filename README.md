# Práctica: Página web con CSS externo

## Objetivo
El objetivo es crear un repositorio local con los archivos de una página web sencilla que incluya la configuración de una hoja de estilos externa (archivo CSS).

## Archivos incluidos
- index.html — Página principal que enlaza al CSS externo.
- styles.css — Hoja de estilos externa que da estilo a la página.

## Cómo usar
1. Abre index.html en tu navegador web (doble clic).
2. (Opcional) Inicia un servidor local, por ejemplo:
   - python -m http.server
   - 
px http-server

## Notas
- Asegúrate de mantener styles.css y index.html en la misma carpeta para que el enlace al CSS funcione correctamente.
